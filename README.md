# Image Search Application

## Overview
This project is a simple yet powerful web application that allows users to search for stunning images using the Pexels API. The application displays image results dynamically based on user queries and supports pagination for viewing more content.

---

## Features
- **Search Functionality:** Type any keyword to fetch relevant images.
- **Responsive Design:** Works seamlessly across different devices.
- **Pagination Support:** Load more results with the "Show More" button.
- **User-Friendly Interface:** Clean and intuitive layout for a great user experience.

## Technologies Used
- **HTML5:** Structure of the app
- **CSS3:** Styling the application
- **JavaScript:** Interactive elements and API handling
- **Pexels API:** Fetching high-quality images

## Setup Instructions
1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Ensure you have an active internet connection for API calls.

## How to Use
1. Enter a search query in the input field (e.g., "Nature").
2. Click the search button to view results.
3. Use the "Show More" button to load additional images.

## API Integration
This app uses the Pexels API. Follow these steps to set up the API:
1. Visit the [Pexels API Documentation](https://www.pexels.com/api/).
2. Sign up and get your API key.
3. Replace `YOUR_API_KEY` in `script.js` with your personal API key.

## Files Overview
- `index.html`: Structure of the application
- `style.css`: Styling for the user interface
- `script.js`: Handles user interactions and API calls

## Future Enhancements
- Add image download functionality
- Display additional image details (photographer, image resolution)
- Include category-based search options
